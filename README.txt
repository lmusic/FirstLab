Данный код обеспечивает доступ к файловой системе компьютера, на котором запущен веб сервер.
Для корректной работы необходимо перед запуском в файле FirstApp.py переменной SERV_DIR присвоить путь до корневой папки сервера. Например: "C:/MyServFolder"

Имеются следующие возможности: 
	1. Предоставление файловой структуры внутри директории. Для этого необходимо в адресной строке прописать путь к директории. Например "http://localhost:8000/oneMoreFolder"
	по этой команде в ответ клиенту вернется список всех элементов содержащихся в папке "C:/MyServFolder/oneMoreFolder"
	2. Скачивание имеющегося элемента. Для этого необходимо в адресную строку дописать название элемента. Например "http://localhost:8000/oneMoreFolder/doc.txt"
	после этого начнется автоматическое скачивание файла doc.txt
	3. Создание/удаление директории в корневой папке сервера. Для этого необходимо после указания пути в котором необходимо создать/удалить директорию передать
	параметр create/delete со значением в виде имени дирректории которую необходимо создать/удалить. Например "http://localhost:8000/oneMoreFolder/?create=anotherFolder" или
	"http://localhost:8000?delete=oneMoreFolder"
